## Setup

```bash
sudo apt update
sudo apt install git curl
sudo curl -fsSL https://deb.nodesource.com/setup_12.x | sudo -E bash -
sudo apt install nodejs build-essential libssl-dev
sudo npm i -g yarn
yarn
```

## Running the app

```bash
yarn dev
```
